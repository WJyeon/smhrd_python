주피터 노트북 사용 방법¶

  • 셀(cell) : 코드 및 텍스트를 입력하는 공간
  • command mode, edit mode
    • command mode : 앞쪽의 띠가 파란색(셀 단위의 작업을 할 수 있는 모드)
    • edit mode: 앞쪽의 띠가 초록색(셀 내부의 텍스트나 코드를 편집할 수 있는 모드)
  • markdown : 셀 하나를 통째로 코드의 영향을 받지 않는 메모장 형태로 만들 때 사용
    • 사진 첨부 가능

  • window + shift + s : 캡처

  • 문자열은 처음부터 끝까지 인덱스 번호를 배정받는다
      

    • 주의1. 인덱스는 0번부터 시작
      • 

    • 주의2. 띄어쓰기도 인덱스 번호를 가진다
      • 

    

    • 

  • 인덱싱: 문자열 안에서 내가 원하는 데이터 하나를 뽑아올 때 사용
      

    • 인덱싱 방법: 변수명[인덱스 번호]
      • 

    

    • 

  • 슬라이싱: 문자열 안에서 범위를 주고 잘라올 때 사용
      

    • 슬라이싱 방법: 변수명[시작 인덱스번호 : 끝 인덱스번호+1]
